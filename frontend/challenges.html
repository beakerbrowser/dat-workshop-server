Home Agenda Challenges Resources

Not-so-hard

  • Open beaker://library/dat://beakerbrowser.com and inspect the source
  • Create a peer-to-peer website that says "Hello, world!"
  • Create a website with a title and description. (See dat.json)
  • Find a website you like and seed it
  • Create a markdown website
  • Create a website and set a custom 404 page

Harder

  • Use HTML and CSS to make your homepage, Geocities style!
  • Do you have a static website? Port your website to `dat://` and publish it under your dat://${yourdomain}.com
  • Use DatArchive.selectArchive() to open Beaker's archive picker
  • Create a text-sharing app! (See paste-dat for inspiration)
  • Use Beaker's DatArchive API to write an hello.txt file that says "Hello, world!"

Hardest

  • Read Beaker's logo using the DatArchive API, then render it into a page
  • Make a simple photo-sharing application (See this example for inspiration)
  • Take a screenshot of a dat:// website using the experimental capturePage API
  • Make a self-editing site that saves changes by writing to itself
  • Dynamically generate content using a custom 404 page.
